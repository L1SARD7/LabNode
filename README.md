# GamePedia - Система обліку та оцінювання відеоігор

## Опис ідеї
Цей проект присвячений створенню веб-додатку для каталогізації відеоігор. Користувачі зможуть переглядати список ігор, дізнаватися деталі (жанр, розробник, рік випуску), а також залишати свої відгуки та оцінки.

**Мета:** Створити зручну платформу для геймерів, де можна знайти цікаві ігри та поділитися думкою.

## Технологічний стек
Для реалізації проекту обрано сучасний стек технологій:

* **Node.js & Express**: Швидка та гнучка платформа для створення REST API.
* **TypeScript**: Для надійності коду та зручності розробки завдяки статичній типізації.
* **PostgreSQL**: Потужна об'єктно-реляційна система керування базами даних. Використовується для надійного зберігання структурованих даних (ігри, користувачі, відгуки).
* **Docker & Docker Compose**: Для контейнеризації середовища розробки. Це дозволяє швидко розгорнути базу даних та додаток на будь-якій машині однією командою.
* **EJS**: Шаблонізатор для рендерингу сторінок на стороні сервера.
